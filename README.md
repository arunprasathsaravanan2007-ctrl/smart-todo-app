# 📝 Smart Todo App

A simple and responsive To-Do List application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by:
  - All
  - Active
  - Completed
- Automatically saves tasks using Local Storage
- Responsive and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How to Use

1. Open the application in your browser.
2. Enter a task in the input field.
3. Click the **Add** button to create a task.
4. Use **Done**, **Edit**, and **Delete** buttons to manage tasks.
5. Use the filter buttons to view All, Active, or Completed tasks.
